# 802.11b Protocol Deep Dive

## Background

802.11b was the first widely deployed Wi-Fi standard, operating in the 2.4 GHz ISM band. It uses HR/DSSS (High Rate Direct Sequence Spread Spectrum) technology, supporting four data rates: 1, 2, 5.5, and 11 Mbps.

## Technical Approach

- **Modulation**: DBPSK (1 Mbps) / DQPSK (2 Mbps) / CCK (5.5/11 Mbps)
- **Spreading**: Barker code (1/2 Mbps) / CCK codewords (5.5/11 Mbps)
- **Channel bandwidth**: 22 MHz
- **Channel plan**: 14 channels (5 MHz spacing), only 3 non-overlapping

## Protocol Analysis

### PPDU Frame Structure

802.11b defines two PPDU formats: Long Preamble and Short Preamble. The main fields include:
- SYNC (synchronization)
- SFD (Start Frame Delimiter)
- SIGNAL (rate/length)
- SERVICE
- PSDU

### Key Technical Points

1. **CCK Modulation**: 5.5 Mbps uses 4 CCK codewords (4 bits each), 11 Mbps uses 64 CCK codewords (8 bits each)
2. **Receiver Sensitivity**: 11 Mbps requires -76 dBm, 1 Mbps requires -94 dBm
3. **Carrier Frequency Tolerance**: ±25 ppm
