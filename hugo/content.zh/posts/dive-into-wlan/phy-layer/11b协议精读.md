# 802.11b 协议精读

## 背景

802.11b 是 Wi-Fi 的第一个广泛部署的标准，工作在 2.4GHz ISM 频段，采用 HR/DSSS（高速直接序列扩频）技术，支持 1/2/5.5/11 Mbps 四种速率。

## 技术路线

- **调制方式**: DBPSK (1Mbps) / DQPSK (2Mbps) / CCK (5.5/11Mbps)
- **扩频**: Barker 码 (1/2Mbps) / CCK 码字 (5.5/11Mbps)
- **信道带宽**: 22MHz
- **信道划分**: 14 个信道（5MHz 间隔），仅 3 个非重叠

## 协议分析

### PPDU 帧结构

802.11b 的 PPDU 分为 Long Preamble 和 Short Preamble 两种格式，主要包含：
- SYNC（同步）
- SFD（起始帧定界）
- SIGNAL（速率/长度）
- SERVICE
- PSDU

### 关键技术点

1. **CCK 调制**: 5.5Mbps 使用 4 个 CCK 码字（每个承载 4bit），11Mbps 使用 64 个 CCK 码字（每个承载 8bit）
2. **接收灵敏度**: 11Mbps 要求 -76dBm，1Mbps 要求 -94dBm
3. **载波频率容限**: ±25ppm
